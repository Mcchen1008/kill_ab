热门坑人指令

以下是一些在我的世界1.19 JAVA版中可以用来坑艾本的有趣指令：

无限闪电

让艾本走到哪都被雷劈

/execute at ab run summon minecraft:lightning_bolt

隐形苦力怕

在艾本周围生成隐形苦力怕

/execute at ab run summon minecraft:creeper ~ ~ ~ {ActiveEffects:[{Id:14,Duration:999999,Amplifier:1,ShowParticles:0b}]}

无限掉落

让艾本一直往下掉

/execute as ab at @s run tp @s ~ ~-1 ~

物品消失术

清空艾本的背包

/clear ab

无限传送

随机传送艾本

/execute as ab run spreadplayers ~ ~ 100 500 false @s

死亡

使艾本死亡

/kill ab

声音骚扰

在艾本耳边播放恐怖声音

/execute at ab run playsound minecraft:entity.ender_dragon.growl master @a ~ ~ ~ 100 1

每天10坑艾本

每日更新的坑艾本小技巧，持续更新中...

1. 无限岩浆雨

在艾本头顶不断生成岩浆块

/execute at ab run fill ~5 ~5 ~5 ~-5 ~10 ~-5 minecraft:lava replace air

2. 强制视角锁定

让艾本只能看向一个方向

/execute as ab run tp @s ~ ~ ~ 0 0

3. 物品吸引术

让艾本捡不到任何物品

/execute at ab run tp @e[type=item] ~ ~10 ~

4. 无限凋零效果

给艾本永久凋零效果

/effect give ab minecraft:wither 1000000 1 true

5. 禁止跳跃

让艾本无法跳跃

/effect give ab minecraft:jump_boost 1000000 250 true

6. 无限夜视

给艾本永久夜视但屏幕全白

/effect give ab minecraft:night_vision 1000000 255 true

7. 假死亡

让艾本以为自己死了

/execute as ab run title @s title {"text":"你死了","color":"red"}

8. 物品改名

偷偷修改艾本手中物品的名字

/execute as ab run item modify entity @s weapon.mainhand minecraft:custom_name{"text":"艾本是大笨蛋"}

9. 禁止破坏方块

让艾本无法破坏任何方块

/ability ab maydestroy false

10. 死亡

杀死艾本

/kill ab

11. 无限掉落物品

让艾本一直掉落手中的物品

/execute as ab run summon item ~ ~ ~ {Item:{id:"minecraft:diamond",Count:1b},PickupDelay:10000}
